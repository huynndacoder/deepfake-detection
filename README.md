## Problem Statement

With the rapid advancement of generative models such as StyleGAN3, synthetic
face images have become nearly indistinguishable from real ones. This project
builds and compares multiple machine learning approaches — from a classical
SVM baseline to fine-tuned deep neural networks — to automatically detect
AI-generated faces.

## Dataset

- **Source:** Kaggle — 10,000 Real vs Fake Faces (StyleGAN3)
- **Generator:** StyleGAN3
- **Size:** ~10,000 images
- **Split:** 70% train / 15% validation / 15% test (stratified)
- **Raw images and trained model weights are not tracked by Git** (see `.gitignore`).
  Download the dataset from Kaggle and place it under `data/raw/` before running any notebook.

## Setup & Installation

```bash
# 1. Clone the repository
git clone https://github.com/huynndacoder/deepfake-detection.git
cd deepfake-detection

# 2. Create and activate a virtual environment
python -m venv .venv
source .venv/bin/activate          # macOS / Linux
# .venv\Scripts\activate           # Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Download the dataset from Kaggle and place it as:
#    data/raw/real faces/  and  data/raw/fake faces/
```
